# spain-open-data
A list of Spanish open data resources. Una lista de datos abiertos de España

The goal of this list is to have a live directory of resources provided by public (add private sector?) administrations. It is organized by country (Spain), autonomous community (i.e Andalucía), province (Sevilla) and finally by city (Sevilla City), sorted alphabetically. 

El objectivo de esta lista es tener un directorio vivo de recursos abiertos de la administración pública española (sector privado también?). Está organizada por país (España), comunidad autónoma (e.g.Andalucía), provincia (e.g.Sevilla) y finalmente por ciudad (ciudad de Sevilla), ordenado de forma alfabética.

Data type / Tipos de datos:
- :bus: Transport / Transporte
- :umbrella: Weather / Clima
- :bar_chart: Statistics / Estadísticas
- :earth_africa: Cartography / Cartografía
- :books: Edu / Educación
- :floppy_disk: General Data / Datos Generales
- :octocat: Code / Programación
- :ocean: Turism / Turismo
